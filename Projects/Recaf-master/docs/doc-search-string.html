Searching for strings

To open string search, select "Search > Strings". This will open a string search window.

Note: In all search types, you will be able to change the string matching mode, and skipped packages

Tips

1. Don't forget about regex

It can be a very powerful tool. In Recaf, the entire string must be matched by the regex pattern to be returned as a result.

Examples:

  • URL: (www|http(s)?:|ftp:)+[^\s]+[\w]
  • IP: (\d{1,3}\.){3}\d{1,3}
  • More examples at regular-expressions.info
  • Check out regex101.com to craft your own patterns with useful guidelines
